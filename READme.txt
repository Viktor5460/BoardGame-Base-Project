Три файла относящихся к классу BoardGame:
BoardGame.h - заголовочный
BoardGame.cpp - определение конструкторов, методов и тп…
main.cpp - тесты класса 


Compile.bat - программа для компиляции //по сути написано g++ -O2 main.cpp BoardGame.cpp -o board_game_test.exe в терминале
run.bat - запуск .exe //board_game_test.exe
